## 遊戲規則
消耗積分參與遊戲，每局遊戲有5次機會，目標是猜出1~100中的終極密碼，越快猜中獲得的積分越多

## 指令
積分: 查詢自己的積分

排行榜: 查詢全部人的積分

進入遊戲: 開始一局新的遊戲

放棄: 放棄當前遊戲

FSM: 顯示FSM Diagram
